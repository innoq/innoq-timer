# INNOQ Timer

Ein Timer im INNOQ Stil, den man bei Remote Trainings in den Pausen zeigen kann

![](img/product.png)

## Vorraussetzungen

- Node.js

## Starten
 1. Dependencies installieren
    ```bash 
    npm ci
    ```
2. Anwendung starten:
   ```bash
   npm run dev
   ```
